# Spring Boot MongoDB Employee Management

This project is a Spring Boot application using MongoDB and Thymeleaf
for managing employees, departments, and projects.

## Features
- Employee CRUD operations
- Thymeleaf-based UI
- MongoDB integration
- Bootstrap styling

## Known Issues
- Employee form throws a Thymeleaf binding error for `projectNameString`
- Issue related to JavaBean getter/setter binding
- Will be fixed in a future update

## Tech Stack
- Spring Boot 3
- MongoDB
- Thymeleaf
- Bootstrap 5
- Java 21
